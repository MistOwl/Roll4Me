ENGLISH
This discord bot is meant to help fiction writers, RPG game masters and artists with inspiration
by radomly picking details for characters, places, events and objects from tables.

COMMANDS: 
/rolltable : You can write the name of a table to get a random item from it. 
ex: by writing '/rolltable Prop' the bot will pick a random prop, like "Ornate Ring"

/showtables : The bot answers the chat with the name of all avaliable tables you can roll in. Use any name showed on the '/rolltable' command! 

/rollhelp : Explains how to use the bot.

/rollcharacter : The bot gives you basic random details for a character, like their flaw, struggles and story hook.


WARNING!!: To use the bot you will need to generate your own discord bot token. Please while creating a bot token allow the general permissions needed for it to read user messages and all text related permission. 
Please paste your Discord Bot Token ID on the appropriate place on tokenDeposit.py

PORTUGUÊS(BR)
Esse bot do discord é feito para ajudar escritores de ficção, mestres de jogo de RPG e artistas com inspiração
ao escolher aleatoriamente detalhes para personagens, lugares, eventos e objetos a partir de tabelas.

COMANDOS: 

/rolltable : Você pode escrever o nome de uma tabela para obter um item dela.
exemplo: Escrevendo '/rolltable Prop' o bot irá selecionar um detalhe ou objeto aleatorio, como um "Anel Ornamentado".

/showtables : O bot responde no chat com o nome de todas as tabelas disponiveis para você rolar. Você utiliza qualquer nome mostrado na tabela no comando '/rolltable'.

/rollhelp : Explica como utilizar o bot.

/rollcharacter : O bot lhe da detalhes básicos aleatorios para um personagem, como suas falhas, lutas e um gancho narrativo.


AVISO!!: Para usar o bot você precisará gerar o seu próprio token de bot de discord. Por favor, enquanto estiver gerando o token de bot, habilite as permissões gerais necessárias para ele ler mensagens de usuário e todas as permissões relacionadas a texto.
Por favor cole o ID do seu token no local apropriado no arquivo tokenDeposit.py
